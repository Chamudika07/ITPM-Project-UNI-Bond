🚀 IMMEDIATE NEXT STEPS (3 Simple Steps!)

STEP 1: Create PostgreSQL Database
───────────────────────────────────

Open PowerShell and run:

   cd backend
   python setup_database.py

When prompted, enter your PostgreSQL admin password (postgres user).

This creates:
  • Database: uni_Bond
  • User: uni_bond_user  
  • Password: uni_bond_secure_password_2024


STEP 2: Start Backend Server
─────────────────────────────

Open Terminal 1 and run:

   cd backend
   .\venv\Scripts\Activate.ps1
   uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

You'll see:
   ✅ INFO:     Application startup complete
   ✅ http://0.0.0.0:8000


STEP 3: Start Frontend Server
──────────────────────────────

Open Terminal 2 (new terminal) and run:

   cd frontend\uniBond_Frontend
   npm run dev

You'll see:
   ✅ Local: http://localhost:5173/
